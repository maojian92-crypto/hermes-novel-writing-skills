# Anti-AI 模式检测指南

> 主服务: 章节写作 Step 2、审查 Step 4
> 内容层级: 缺陷补偿层（网文特定防AI味协议）

---

## 一、AI味五层检测模型

### 1. 词汇层

**高频AI词汇清单（每500字出现上限）：**

| 词汇/模式 | 上限 | 替代方案 |
|-----------|------|----------|
| 缓缓+动词 | ≤1 | 直接写动作，不加副词 |
| 淡淡+动词 | ≤1 | 用具体表情/动作替代 |
| 微微+动词 | ≤1 | 精确描述幅度 |
| 轻轻+动词 | ≤1 | 具体力度描写 |
| 眸中闪过 | ≤1 | 具体眼神变化 |
| 瞳孔微缩 | ≤1 | 生理反应+心理 |
| 心中一凛 | ≤1 | 身体反应+决策 |
| 嘴角微微上扬 | ≤1 | 笑容的具体形态 |

**检测命令：**
```bash
grep -oE '缓缓|淡淡|微微|轻轻|眸中闪过|瞳孔微缩|心中一凛|嘴角微微上扬' 第XXX章-标题.md | wc -l
```

### 2. 句式层

**禁止模式：**

**A. "起因→经过→结果→感悟"四段闭环**
```
错误示例：
他走进房间（起因），看到桌上有一封信（经过），
打开发现是挑战书（结果），他终于明白敌人已经盯上他了（感悟）。

正确：
他推开房门，目光落在桌上——那里有一封信，火漆印是黑色的。
他拆开，只有一行字："子时，城西。"
```

**B. 连续同构句（≥3句主谓宾一致）**
```
错误示例：
他走进了房间。他拿起了信件。他拆开了信封。

正确：
他推门而入。桌上的信件在月光下泛着白。他拿起来，火漆碎裂。
```

**C. 每段总结句收尾**
```
错误示例：
……（一段描写）
他终于明白了真相。
……（一段对话）
由此可见，敌人早有预谋。

正确：
……（一段描写）
……（直接开始下一段，不总结）
```

**D. 同一信息重复说**
```
错误示例：
他很强大。没有人能打败他。他的实力是顶级的。

正确：
只说一次，用场景证明。
```

### 3. 叙事层

**禁止模式：**

**A. 节奏匀速**
```
错误：每段都是4-5句，每句都是15-20字。
正确：有的段落一句话，有的段落一大段。快慢交替。
```

**B. 戏剧性反讽提示**
```
错误："他不知道的是，危险正在逼近。"
正确：直接写危险逼近的场景，不提醒读者。
```

**C. 章末安全着陆**
```
错误："一切都结束了，他们终于安全了。"
正确："门在他身后关上，但他注意到——锁是开着的。"
```

**D. 展示后解释**
```
错误：
他握紧拳头（展示）。
这说明他很愤怒（解释）。

正确：
他握紧拳头，指节发白。
```

### 4. 情感层

**禁止标签化情绪：**

| 标签化写法 | 行为暗示替代 |
|-----------|-------------|
| 他感到愤怒 | 他握紧拳头，指节发白 |
| 她非常紧张 | 她反复摩挲袖口 |
| 他很悲伤 | 他盯着地面，很久没眨眼 |
| 她很害怕 | 她后退一步，肩膀抵住墙 |

**禁止情绪即时切换：**
```
错误：
他很愤怒。但他立刻平静下来。

正确：
他很愤怒。他深吸一口气，数到三，才开口说话——声音 still 有点抖。
```

**禁止全员同一反应模板：**
```
错误：所有人都是"瞳孔微缩""心中一凛"。
正确：暴躁者直接开骂，冷静者沉默，胆小者后退。
```

### 5. 对话层

**禁止信息宣讲：**
```
错误：
"这是什么？"
"这是X，它是Y年前由Z发明的，主要用于A功能。"

正确：
"这是什么？"
"别碰。"他挡在她前面，"上次碰过这东西的人，现在还在医院里。"
```

**禁止全员书面语：**
```
错误：
角色A："此事恐怕不妥。"
角色B："确实如此，依我之见……"
角色C："所言极是，然则需考虑……"

正确：
角色A（粗人）："这不行。"
角色B（学者）："从理论上说……"
角色C（少年）："为啥不行？我觉得挺好啊。"
```

**禁止对白后解释：**
```
错误：
"我不去。"他说。他这么说是因为害怕。

正确：
"我不去。"他说，手指攥紧了背包带。
```

## 二、AI味修复速查表

| 问题 | 快速修复 |
|------|----------|
| "缓缓" | 删掉，直接写动作 |
| "淡淡" | 删掉，写具体表情 |
| "他感到X" | 改为身体反应+动作 |
| "他终于明白了" | 删掉，直接写后续行动 |
| "殊不知" | 删掉，切换视角写另一方 |
| "像……像……像……" | 只保留一个比喻，展开描写 |
| 连续同构句 | 改变句式结构，长短交替 |
| 每段总结 | 删掉总结句，直接过渡 |
| 信息宣讲对话 | 加潜台词、动作、打断 |
| 全员书面语 | 给每个角色加口语标记 |

## 检测命令汇总

```bash
# 万能副词（阈值：500字内≤1次，全章累计建议≤3次）
grep -oE '缓缓|淡淡|微微|轻轻' 第XXX章-标题.md | wc -l

# 神态模板（阈值：≤2处/章）
grep -oE '眸中闪过|瞳孔微缩|心中一凛|嘴角微微上扬' 第XXX章-标题.md | wc -l

# 情绪标签（阈值：≤2处/章）
grep -oE '他感到|她感到|感到非常|感到一阵' 第XXX章-标题.md | wc -l

# 总结句（阈值：≤2处/章）
grep -oE '他终于明白了|他终于知道|由此可见|这说明' 第XXX章-标题.md | wc -l

# 反讽提示（阈值：≤2处/章）
grep -oE '他不知道的是|殊不知|没想到|没料到' 第XXX章-标题.md | wc -l

# 安全着陆（阈值：≤1处/章，建议为0）
grep -oE '一切都结束了|终于安全了|完美解决|尘埃落定' 第XXX章-标题.md | wc -l

# 展示后解释（阈值：≤2处/章）
grep -oE '这说明|这意味着|这表明|这代表' 第XXX章-标题.md | wc -l

# 句式模板（致命项，必须为0）
# 是……停顿（引号内）
grep -oE '"[^"]*是……[^"]*。"' 第XXX章-标题.md | wc -l
# 是……停顿（无引号叙述）
grep -oE '，是……|。是……|：是……' 第XXX章-标题.md | wc -l
# 不是……是……
grep -oE '不是[^，。]*是……' 第XXX章-标题.md | wc -l
# 排比比喻
grep -oE '像[^，]*，像[^，]*，像' 第XXX章-标题.md | wc -l
# 模糊指代
grep -oE '某种……' 第XXX章-标题.md | wc -l
# 否定排比
grep -oE '不是[^，]*，不是[^，]*，不是' 第XXX章-标题.md | wc -l
# 扩散句式
grep -oE '从[^，]*，到[^，]*。从[^，]*，到' 第XXX章-标题.md | wc -l
```

**阈值标准（统一自33维审计）：**
- 万能副词：500字内≤1次，全章累计建议≤3次
- 神态模板：≤2处/章
- 情绪标签：≤2处/章
- 总结句：≤2处/章
- 反讽提示：≤2处/章
- 安全着陆：建议为0
- 展示后解释：≤2处/章
- 任何单项严重超标 → 需要修复
- 万能副词+神态模板+情绪标签合计>10处 → 需要全面检查
