# 六维审查清单

> 主服务: 章节写作后自检
> 内容层级: 质量闸门

---

## 审查维度

### 1. 设定一致性（Consistency）

**检查项：**
- [ ] 角色能力是否与当前状态匹配
- [ ] 地点描述是否与世界观一致
- [ ] 物品/规则使用是否符合已建立设定
- [ ] 新实体是否有合理来源（非凭空创造）

**检查方法：**
```bash
# 对比 Truth Files
grep -n "角色名" 02-团队成员状态跟踪.md
grep -n "地点名" 05-世界观与规则文档.md
```

**严重度：**
- critical: 角色能力超越设定且无解释
- high: 地点描述与已建立规则矛盾
- medium: 物品使用方式轻微偏差

---

### 2. 时间线（Timeline）

**检查项：**
- [ ] 本章时间是否与上章衔接（无回跳）
- [ ] 倒计时/截止日期是否正确推进
- [ ] 无角色同时出现在两个地点
- [ ] 跨夜/跨日有过渡描写

**检查方法：**
```bash
# 查看上章时间
grep -n "时间\|夜晚\|白天\|次日" 第XXX-1章-标题.md | tail -5
# 查看本章时间
grep -n "时间\|夜晚\|白天\|次日" 第XXX章-标题.md | head -5
```

**严重度：**
- critical: 时间回跳且无解释
- high: 角色同时出现在两个地点
- medium: 时间推进模糊

---

### 3. 叙事连贯（Continuity）

**检查项：**
- [ ] 上章钩子是否有回应
- [ ] 场景转换是否有过渡
- [ ] 情绪弧是否连续（上章愤怒本章突然平静无过渡）
- [ ] 伏笔是否有推进或回收

**检查方法：**
```bash
# 查看上章结尾
tail -20 第XXX-1章-标题.md
# 查看本章开头
head -20 第XXX章-标题.md
```

**严重度：**
- critical: 上章明确承诺完全未回应
- high: 情绪断裂无过渡
- medium: 场景转换生硬

---

### 4. 角色一致性（Character）

**检查项：**
- [ ] 对话风格是否符合角色特征
- [ ] 行为是否与已建立的性格/动机一致
- [ ] 角色知识边界合理（不使用不应知道的信息）
- [ ] 口吻差异化（非全员同一语气）

**检查方法：**
```bash
# 检查角色对话
grep -n '"' 第XXX章-标题.md | grep "角色名"
```

**口吻标准：**

| 角色类型 | 句长 | 词汇标记 | 示例 |
|---------|------|---------|------|
| 暴躁少年 | 短句，断句多 | 口语化，带语气词 | "滚！别废话！" |
| 老学究 | 长句，从句多 | 文言痕迹，引经据典 | "此事……若依老夫所见，恐怕不妥。" |
| 市井商人 | 中等，话术感 | 客套+算计 | "张爷您说笑了，小本生意，哪敢怠慢。" |
| 冷静领袖 | 中等，克制 | 理性分析，少情绪词 | "先确认损失，再决定下一步。" |

**严重度：**
- critical: 角色行为与核心人设矛盾
- high: 对话风格偏离角色特征
- medium: 知识边界轻微越界

---

### 5. 逻辑（Logic）

**检查项：**
- [ ] 因果关系是否成立
- [ ] 角色决策是否有合理动机
- [ ] 冲突结果是否符合力量对比
- [ ] 无"降智"行为（角色突然变蠢推动剧情）

**检查方法：**
```bash
# 检查决策段落
grep -n "决定\|选择\|因为\|所以" 第XXX章-标题.md
```

**严重度：**
- critical: 核心因果关系不成立
- high: 角色决策无合理动机
- medium: 力量对比轻微失衡

---

### 6. AI味检测（AI Flavor）

**检查项：**

#### 6.1 词汇层
- [ ] 无高频AI词汇密集
- [ ] "缓缓/淡淡/微微"在500字内<=1次，全章累计建议<=3次
- [ ] "眸中闪过""瞳孔微缩"等神态模板<=2次/章

#### 6.2 句式层
- [ ] 无"起因->经过->结果->感悟"四段闭环
- [ ] 无连续同构句（>=3句主谓宾一致）
- [ ] 无每段总结句收尾（"他终于明白了/由此可见"<=2处/章）

#### 6.3 叙事层
- [ ] 节奏有快慢变化（非匀速）
- [ ] 无"他不知道的是……"戏剧性反讽（<=2处/章）
- [ ] 章末非"安全着陆"（保留未解决问题，建议为0）
- [ ] 无展示后紧跟解释（"这说明/这意味着"<=2处/章）

#### 6.4 情感层
- [ ] 情绪用行为暗示，无标签化（"他感到X"<=2处/章）
- [ ] 无情绪即时切换（有过渡）
- [ ] 非全员同一反应模板

#### 6.5 对话层
- [ ] 对话带意图冲突，非信息宣讲
- [ ] 非全员书面语
- [ ] 无对白后解释性叙述

**检测命令：**
```bash
# 万能副词（阈值：500字内≤1次，全章累计建议≤3次）
grep -oE '缓缓|淡淡|微微|轻轻' 第XXX章-标题.md | wc -l

# 神态模板（阈值：≤2处/章）
grep -oE '眸中闪过|瞳孔微缩|心中一凛|嘴角微微上扬' 第XXX章-标题.md | wc -l

# 情绪标签（阈值：≤2处/章）
grep -oE '他感到|她感到|感到非常|感到一阵' 第XXX章-标题.md | wc -l

# 总结句（阈值：≤2处/章）
grep -oE '他终于明白了|他终于知道|由此可见|这说明' 第XXX章-标题.md | wc -l

# 反讽提示（阈值：≤2处/章）
grep -oE '他不知道的是|殊不知|没想到|没料到' 第XXX章-标题.md | wc -l

echo "安全着陆（建议为0）:"
grep -oE '一切都结束了|终于安全了|完美解决|尘埃落定' 第XXX章-标题.md | wc -l
echo "展示后解释（<=2处/章）:"
grep -oE '这说明|这意味着|这表明|这代表' 第XXX章-标题.md | wc -l
```

**严重度：**
- high: AI词汇密集（超过阈值）
- high: 句式模板化严重
- medium: 个别AI味表达

---

## 审查输出格式

```markdown
## 审查报告：第XXX章

### 设定一致性
- 状态: 通过 / 问题 / 阻断
- 问题: [如有]

### 时间线
- 状态: 通过 / 问题 / 阻断
- 问题: [如有]

### 叙事连贯
- 状态: 通过 / 问题 / 阻断
- 问题: [如有]

### 角色一致性
- 状态: 通过 / 问题 / 阻断
- 问题: [如有]

### 逻辑
- 状态: 通过 / 问题 / 阻断
- 问题: [如有]

### AI味检测
- 状态: 通过 / 问题 / 阻断
- 问题: [如有]

### 总结
- 阻断问题数: X
- 高优问题数: Y
- 建议修复: [列表]
```

---

## 阻断规则

- 存在任何 critical 问题 -> 必须修复后才能继续
- 存在 3+ 个 high 问题 -> 建议修复
- AI味检测 high -> 必须修复（影响阅读体验）

---

## 快速自检命令

```bash
# 一键运行所有检测
echo "=== 万能副词 ==="
grep -oE '缓缓|淡淡|微微|轻轻' 第XXX章-标题.md | wc -l
echo "=== 神态模板 ==="
grep -oE '眸中闪过|瞳孔微缩|心中一凛|嘴角微微上扬' 第XXX章-标题.md | wc -l
echo "=== 情绪标签 ==="
grep -oE '他感到|她感到|感到非常|感到一阵' 第XXX章-标题.md | wc -l
echo "=== 总结句 ==="
grep -oE '他终于明白了|他终于知道|由此可见|这说明' 第XXX章-标题.md | wc -l
echo "=== 反讽提示 ==="
grep -oE '他不知道的是|殊不知|没想到|没料到' 第XXX章-标题.md | wc -l
echo "安全着陆（建议为0）:"
grep -oE '一切都结束了|终于安全了|完美解决|尘埃落定' 第XXX章-标题.md | wc -l
echo "展示后解释（<=2处/章）:"
grep -oE '这说明|这意味着|这表明|这代表' 第XXX章-标题.md | wc -l
echo "=== 句式模板（致命项） ==="
echo "是……停顿（引号内）:"
grep -oE '"[^"]*是……[^"]*。"' 第XXX章-标题.md | wc -l
echo "是……停顿（无引号叙述）:"
grep -oE '，是……|。是……|：是……' 第XXX章-标题.md | wc -l
echo "不是……是……:"
grep -oE '不是[^，。]*是……' 第XXX章-标题.md | wc -l
echo "排比比喻:"
grep -oE '像[^，]*，像[^，]*，像' 第XXX章-标题.md | wc -l
echo "否定排比:"
grep -oE '不是[^，]*，不是[^，]*，不是' 第XXX章-标题.md | wc -l
echo "扩散句式:"
grep -oE '从[^，]*，到[^，]*。从[^，]*，到' 第XXX章-标题.md | wc -l
echo "模糊指代:"
grep -oE '某种……' 第XXX章-标题.md | wc -l
echo "=== 字数 ==="
grep -oP '[\x{4e00}-\x{9fff}]' 第XXX章-标题.md | wc -l
```
